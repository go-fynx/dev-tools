Cron utilities

Standard five-field cron (minute, hour, day-of-month, month, day-of-week). Day-of-week uses 0 = Sunday … 6 = Saturday. All logic runs in your browser.

Cron builder

Pick a schedule type and fill the fields. The expression and a short description update as you go.

With Every N hours set to 1, the job runs at the chosen minute in every hour (same as classic “hourly”). Larger values use the hour column step (e.g. minute 0 every 2 hours → 0 */2 * * *).

Days of week

Cron → human

Paste a five-field expression. Spaces separate fields; common patterns get a short English phrase.

Programmatic API

Functions live on window.cronUtils (see js/cron-utils.js).

// Weekly: Monday 09:00 → "0 9 * * 1"
cronUtils.generateCron({
  type: 'weekly',
  day: [1, 2, 3, 4, 5],
  hour: 9,
  minute: 0
});

cronUtils.cronToHuman('*/5 * * * *');  // "Every 5 minutes"
cronUtils.cronToHuman('0 9 * * 1');     // "Every Monday at 09:00"
cronUtils.cronToHuman('30 14 * * *');   // "Every day at 14:30"
